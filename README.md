# Real Estate Listing & Data Management Automation

This project is a custom-built automation tool developed using **Google Apps Script** to streamline real estate operations. It centralizes property data and automates the archiving process.

## 🚀 Key Features
- **Automated Archiving:** 100% automated status tracking (Active, Passive, Archived).
- **Centralized Database:** A Google Sheets-based system to manage all property listings in one place.
- **Operational Efficiency:** Reduced manual data entry time by approximately **40%**.

## 🛠️ Built With
- **Google Apps Script (JavaScript based)**
- **Google Sheets API**

## 📈 Impact
Designed as a bespoke solution for real estate portfolio management to eliminate data fragmentation and improve reporting speed.
